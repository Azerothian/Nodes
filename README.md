## Install

### [Node](http://nodejs.org/), [Browserify](http://browserify.org/)
- Use: `require('nodes')`
- Install: `npm install --save nodes`

### [Ender](http://ender.jit.su/)
- Use: `require('nodes')`
- Install: `ender add nodes`
